Este projeto é um Ajuda - Faculdade Dunamis, aqui terão manuais, documentos, tutoriais... então deve ter um ahome com principais links e a depender da categoria o layout pode mudar, a princípio o post default tem um layout default, mas os de categoria "manuais" terão um layout com sidemenu à esquerda referente ao conteúdo.

Páginas:
/ (big menus com principais links)/
/manuais/ (lista de manuais) - Layout com sidemenu à esquerda
/tutoriais/ (lista de tutoriais) - Layout default
/documentos/ (lista de documentos) - Layout default
/categoria/x/ (posts de cada categoria) - Layout default

Técnicos

- Use Tailwind CSS
- Deve ser responsivo
- Como é wordpress use blocks para compor o layout
- Deve ser possível editar o conteúdo via editor de blocos do wordpress
- A carga das imagens deve ser lazy load
- Construa ou use plugin para marcar se o conteúdo ajudou ou não
- Configure plugins de SEO
- Configure plugins de cache
- Configure plugins de segurança
